---
title: Operasi Crud
---

Pada aplikasi KosFunds, operasi CRUD (Create, Read, Update, Delete) diterapkan dalam berbagai fitur utama untuk memungkinkan pengguna mengelola data secara efektif. Berikut adalah implementasi masing-masing operasi CRUD dalam aplikasi:

## 1. Create
- **Budget**  
    Pengguna dapat menambahkan anggaran baru sesuai dengan kebutuhan.
- **Income**  
    Pengguna dapat mencatat pemasukan baru, termasuk sumber dan jumlahnya.
- **Expense**  
    Pengguna dapat menambahkan pengeluaran baru dengan detail seperti kategori, deskripsi, dan jumlah.
- **Akun**  
    Admin dapat membuat akun pengguna baru melalui fitur manajemen akun.

## 2. Read
- **Dashboard**  
    Menampilkan data total budget, pemasukan, pengeluaran, batas pengeluaran, dan persentase serta laporan pengeluaran keuangan dalam bentuk tabel dan grafik.
- **Detail Spending**  
    Menampilkan daftar transaksi berdasarkan kategori yang dipilih, termasuk waktu transaksi dan jumlahnya.
- **Manajemen Akun (Admin)**  
    Admin dapat melihat daftar akun pengguna yang terdaftar.

## 3. Update
- **Budget**  
    Pengguna dapat memperbarui rincian anggaran, seperti jumlah atau nama.
- **Income**  
    Pengguna dapat mengedit detail pemasukan yang telah dicatat, termasuk sumber dan jumlah.
- **Expense**  
    Pengguna dapat mengubah data pengeluaran, seperti jumlah, kategori, atau deskripsi.
- **Manajemen Akun (Admin)**  
    Admin dapat mengedit informasi akun pengguna, seperti nama, email, atau hak akses (role).

## 4. Delete
- **Budget**  
    Pengguna dapat menghapus anggaran yang tidak lagi relevan.
- **Income**  
    Pengguna dapat menghapus data pemasukan tertentu.
- **Expense**  
    Pengguna dapat menghapus pengeluaran yang salah atau tidak diperlukan lagi.
- **Manajemen Akun (Admin)**  
    Admin dapat menghapus akun pengguna jika diperlukan, misalnya jika akun tersebut tidak aktif atau melanggar kebijakan.
