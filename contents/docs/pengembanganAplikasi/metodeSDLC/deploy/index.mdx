title: Deployment
---

Deployment merupakan tahap penting dalam siklus pengembangan aplikasi, yang bertujuan untuk memindahkan aplikasi dari lingkungan pengembangan ke server produksi agar dapat diakses oleh pengguna secara online. Pada proyek KosFunds, aplikasi dideploy menggunakan layanan hosting Rumahweb, yang menyediakan platform yang stabil dan mudah diakses untuk pengelolaan aplikasi web.

Berikut link website saya: https://kosfunds.fun/
