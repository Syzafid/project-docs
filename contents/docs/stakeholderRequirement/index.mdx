## Internal Stakeholder

**Developer**

Sebagai seorang developer, saya ingin:
- Mendapatkan dokumentasi kebutuhan bisnis yang jelas untuk mengembangkan aplikasi sesuai spesifikasi.
- Menggunakan arsitektur sistem yang modular agar fitur baru mudah ditambahkan di masa depan.
- Memiliki akses ke sistem logging dan monitoring untuk mempermudah debugging dan perbaikan.
- Mendapatkan panduan teknis dan data API untuk integrasi dengan layanan eksternal seperti AI atau otentikasi login pihak ketiga.
- Mengetahui feedback dari pengguna untuk mengoptimalkan performa dan fitur aplikasi.


## Eksternal Stakeholder

**User**

Sebagai seorang pengguna, saya ingin:
- Mencatat pemasukan dan pengeluaran saya dengan mudah dan akurat.
- Mendapatkan rekomendasi keuangan yang relevan berdasarkan kebiasaan finansial saya.
- Membuat dan memantau tujuan keuangan saya agar lebih terorganisir.
- Menggunakan fitur pengingat untuk pembayaran atau pengeluaran mendatang.
- Menyesuaikan tampilan aplikasi sesuai preferensi saya, seperti tema warna dan kategori spending.
- Menikmati pengalaman pengguna yang intuitif, bahkan jika saya bukan orang yang ahli teknologi.
- Memastikan data pribadi saya aman dan tidak disalahgunakan.
