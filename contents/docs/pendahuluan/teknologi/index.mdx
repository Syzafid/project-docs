## Backend Development
Aplikasi ini dibangun menggunakan bahasa pemrograman PHP native untuk mengatur logika dan proses backend, termasuk pengelolaan data dan integrasi dengan database.

## Frontend Development
Untuk tampilan dan styling, digunakan kombinasi CSS dan Bootstrap guna menciptakan antarmuka yang responsif dan estetis.
JavaScript digunakan untuk menambahkan elemen interaktif dan meningkatkan pengalaman pengguna.

## Database Management
Database aplikasi ini dikelola menggunakan MySQL yang diakses melalui phpMyAdmin pada aplikasi XAMPP. Koneksi antara database dan aplikasi diatur menggunakan kode PHP.

## Desain Wireframe dan Prototype
Proses desain awal, seperti wireframe dan prototype, dilakukan menggunakan Figma untuk memberikan gambaran jelas mengenai tampilan dan alur aplikasi.

## Flowchart dan Diagram
Alur proses aplikasi dan diagram pendukung lainnya dirancang menggunakan Draw.io dan Miro untuk memvisualisasikan logika aplikasi secara rinci.

## Kode dan Pengembangan
Selama proses pengkodean, digunakan Visual Studio Code (VSCode) sebagai text editor utama karena kemampuannya yang mendukung pengembangan aplikasi berbasis web secara efisien.
