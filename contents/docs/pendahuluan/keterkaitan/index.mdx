title: Keterkaitan dengan Mata Kuliah

<hr />
Proyek akhir KosFunds ini dirancang dan dikembangkan berdasarkan materi yang telah dipelajari dari beberapa mata kuliah terkait. Berikut adalah penjelasannya:

## Pemrograman Web Lanjut
Proyek ini memanfaatkan pengetahuan dari mata kuliah Pemrograman Web Lanjut, terutama dalam:
- Penggunaan PHP native sebagai backend untuk memproses logika aplikasi dan menghubungkan dengan database.
- Implementasi teknologi frontend seperti Bootstrap, CSS, dan JavaScript untuk membangun antarmuka yang responsif dan interaktif, sesuai dengan praktik terbaik yang dipelajari.

## Praktik Analisis dan Desain Perangkat Lunak
Pada tahap awal proyek, kebutuhan aplikasi dianalisis menggunakan teori wawancara dengan stakeholder, yang merupakan metode yang dipelajari dalam mata kuliah ini. Analisis kebutuhan membantu dalam menentukan fitur yang relevan dan sesuai dengan kebutuhan pengguna.

## Perancangan Antarmuka Pengguna
Desain antarmuka aplikasi ini didasarkan pada materi yang dipelajari di mata kuliah ini, dengan menggunakan:
- Desain bento untuk menyusun layout dashboard secara intuitif.
- Teori pewarnaan monokrom dengan warna hijau neon dan hitam sebagai warna utama, menciptakan tampilan yang menarik sekaligus profesional.

## Basis Data
Pengelolaan data aplikasi menggunakan MySQL sebagai sistem manajemen basis data, sesuai dengan materi yang dipelajari di mata kuliah Basis Data. Relasi antar tabel diterapkan untuk mengelola data transaksi dan pengguna, memastikan integritas dan efisiensi pengelolaan data.

## Metode dan Model Pengembangan Perangkat Lunak
Pengembangan proyek ini mengikuti metode Scrum, yang dipelajari di mata kuliah Metode dan Model Pengembangan Perangkat Lunak. Metode ini memungkinkan tim untuk bekerja secara iteratif dan kolaboratif, memastikan proyek tetap adaptif terhadap perubahan kebutuhan pengguna.
