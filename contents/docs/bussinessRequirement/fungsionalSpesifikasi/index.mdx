## Spesifikasi Kebutuhan Bisnis

KosFunds adalah platform berbasis web yang membantu pengguna mengelola keuangan mereka secara terintegrasi. Platform ini dirancang agar dapat diakses melalui desktop dan mobile browser dengan model B-to-C (Business to Costumer).

Pengguna dapat mengelola anggaran, memonitor pengeluaran, membuat tujuan keuangan, serta menerima rekomendasi keuangan berbasis AI untuk mencapai target finansial mereka.

**Kategori dalam aplikasi KosFunds meliputi:**

**Dashboard Keuangan**
- Total pemasukan, pengeluaran, dan anggaran.
- Grafik pembagian uang berdasarkan kategori spending.

**Menu Spending**
- Rincian pengeluaran berdasarkan kategori seperti makanan, transportasi, dan utilitas.
- Riwayat transaksi

**Goals Keuangan**
- Fitur perencanaan dan monitoring tujuan keuangan spesifik, seperti tabungan atau pembayaran utang.

**Saran Keuangan**
- Rekomendasi otomatis berbasis API Gemini AI sesuai kondisi finansial pengguna.

**Upcoming Spending**
- Pengingat transaksi yang akan datang.
